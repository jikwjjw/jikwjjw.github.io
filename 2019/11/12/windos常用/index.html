windos常用快捷键

  • 显示桌面或恢复桌面

    Windows+D

  • 打开”我的电脑”

    Windows+E

  • 查找文件或文件夹

    Windows+F

  • 开启“运行” 对话框

    Windows+R

  • 开启 “公用程序管理器”

    Windows+U

  • 锁定计算机

    Windows+L

  • 打开的应用程序切换

    Windows+Tab

  • 显示桌面小工具

    Windows+G

  • 打开“外接显示”的设置面板

    Windows+P

  • 使当前使用的窗口变化

    Windows 键 + 方向键“↑”, 方向鍵“↓”,方向键“←”,方向键“→”

  • 打开设置

    Windows+I

  • 打开搜索

    Windows+S

编辑模式

  • 截图 :

    Ctrl+Alt+S

  • 查看电脑最近使用 :

    Windows+R+”recent”
