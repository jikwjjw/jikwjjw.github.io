jupyter使用的快捷方式

  • 安装jupyter notebook

    pip install jupyter notebook

  • Shift-Enter :

    运行本单元，选中下个单元

  • Y:

    单元转入代码状态

  • M:

    单元转入markdown状态

  • R:

    单元转入raw状态

  • K:

    选中上方单元

  • J:

    选中下方单元

  • A:

    在上方插入新单元

  • B:

    在下方插入新单元

  • X:

    剪切选中的单元

  • C:

    复制选中的单元

  • V:

    粘贴到下方单元

  • Z:

    恢复删除的最后一个单元

  • D,D:

    删除选中的单元

  • Shift-M:

    合并选中的单元

  • H:

    显示快捷键帮助

编辑模式

  • Tab :

    代码补全或缩进

  • Shift-Tab :

    提示

  • Ctrl-] :

    缩进

  • Ctrl-[ :

    解除缩进

  • Ctrl-Z :

    复原

  • Esc :

    进入命令模式

  • Ctrl-Shift– :

    分割单元

  • Up :

    光标上移或转入上一单元

  • Down :

    光标下移或转入下一单元

  • Ctrl-Left :

    跳到左边一个字首

  • Ctrl-Right :

    跳到右边一个字首
