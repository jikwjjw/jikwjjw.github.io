Blog搭建

May 06, 2019 1158

blog搭建

下载nodejs

下载git for windos

配置git ssh

配置环境变量 cmd git

新建GitHub仓库 名字为jikwjjw.github.io

E盘新建 blog 文件夹

cd e:/blog

下载hexo hexo官方文档中有下载方式

阿里镜像源：npm config set registry https://registry.npm.taobao.org/

确认是否切换成功：npm config get registry

E:\bolg>npm install -g hexo-cli（可以用阿里的镜像源加速）

E:\bolg>hexo init jikwjjw

E:\bolg>cd jikwjjw

E:\bolg\jikwjjw>npm install

E:\bolg\jikwjjw>hexo s -p 5555 (http://localhost:5555):配置端口

下载Sublime

把blog拖进Sublime中

修改内容

修改source ->_posts -> helloWorld.md即可

部署内容

在hexo文档中有(部署)，修改配置文件说明

修改deploy中的

type : git

repo : https://github.com/jikwjjw/jikwjjw.github.io.git

branch: master

16行 : url : https://jikwjjw.github.io

author : jikwjjw

安装git插件：

E:\bolg\jikwjjw>npm install hexo-deployer-git –save

部署完成运行（可能需要配置github密码等）：

cmd E:\bolg\jikwjjw>hexo g

cmd E:\bolg\jikwjjw>hexo d

Username for 'https://github.com': 输入的是github上的邮箱账号
Password for 'https://你的github邮箱@github.com': 输入github的登录密码,点击enter键即可.

主题修改

在https://hexo.io/themes/ 下载一款主题，解压缩到E:\bolg\jikwjjw\themes中

修改配置文件中的 theme : next 即可

E:\bolg\jikwjjw>hexo s -p 5555 (http://localhost:5555):本地预览

百度hexo next的使用即可
